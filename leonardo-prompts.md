## Batch Generation Workflow

1. Add the **Style Recipe** to your user settings in Leonardo as a saved prompt prefix
2. Copy each preset prompt one at a time
3. Generate 4 variants per preset
4. Download the best one → rename to `preset-{number}-{slug}.png`
5. Place in `simulator/public/scenes/`
